# Anime-Next-Watch
Anime Next Watch is a responsive web application that helps users discover trending anime and decide what to watch next. It integrates real-time data from the Jikan API, providing an interactive and dynamic user experience.

Search Functionality :---
Easily search anime by title using real-time filtering.

Genre Filtering :---
Filter anime based on genres using dropdown selection.

Sorting Options :---
Sort anime by:
  1. Top Rated
  2. Alphabetical (A–Z)

Expandable Synopsis :---
View or hide detailed anime descriptions with a single click.

Dark / Light Mode :---
Toggle between themes for better user experience.

Real-Time API Integration :---
Fetches live anime data dynamically.

Responsive Design :---
Works seamlessly across mobile, tablet, and desktop devices.

Tech Stack :---
1. HTML
2. CSS
3. JavaScript
   
Fetch API :---
Array Higher-Order Functions (map, filter, sort)

Project Goal :---

This project was built to :---

-> Practice working with public APIs
-> Implement dynamic UI updates
-> Use higher-order functions instead of loops
-> Build a responsive and interactive web application

API Used :---
Jikan API
(Provides anime data such as titles, ratings, genres, and synopsis)
